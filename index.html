Vad är ett attribut


Till Google!
href → attribut!

Simpla Attribut Selectorer


img[alt] {
	background-color: #FFF;
}

a[href="http://google.se"] {
	font-size: 25em;
}

Egna attribut


Homepage
					

a[go-to="homepage"] {
	font-size: 3000px;
}

Regex Attribut Selectorer


a[href*="google"] {
	background-color: green;
}

a[href^="http"] {
	background-color: red;
}

img[src$=".jpg"] {
	border-bottom: 5px dotted blue;
}

Tilda (~) selectorn





					

img[title~="vatten"] {
	border: 5px groove blue;
}
img[title~="hus"] {
	border-radius: 5px;
}
